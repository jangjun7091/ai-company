---
name: design-discovery
description: Produce a Design Brief for a mission with a user-facing surface, then execute it by mapping its sections onto existing design skills.
---

Read `AGENTS.md` and the design pass in `company/workflows/idea-to-release.md`; record the result with `templates/design-brief.md`.

When executing the brief, map its sections onto existing skills instead of reinventing them:

- Experience intent and users — `design:user-research`, `design:research-synthesis`
- Shared shell and visual language — `design:design-system`, `theme-factory`, `brand-guidelines`
- Proposals and implementation — `frontend-design`, `canvas-design`
- Verifiable proxies — `design:accessibility-review`; intent alignment via `design:design-critique`
- Interface copy — `design:ux-copy`
- Handoff to build tasks — `design:design-handoff`
